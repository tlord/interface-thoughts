### Just enter it, you worm

(Original tweet: https://x.com/timothylord/status/1997879583987933635)

This pop-up (with no obvious way to kill it, to reject it permanently, or even to put it off like a software upgrade) would pop up three times every time I logged in to my Mac.

[IMAGE: enter-apple-password-001.png]

No link to explain what "Apple Media Services" are or do. No explanation of whether or how I might be using them at present, or why I should care to. The attitude it projects is indifferent arrogance -- unnecessarily so. 

##How to fix it##

1) Make it optional and polite, instead of insistent and presumptuous. Invert the order of the words, add a checkbox and button to continue. 

"To continue using Apple Media Services on this Mac, please enter your Apple ID password." 

That's straightforward. 

Even better, add a line of Why: 

"Apple Media Services are important if you use FaceTime and certain other Apple applications."

2) Be honest. If there's a Cancel button, the same message shouldn't pop up as soon as the user chooses to cancel. This is one place where an "Ask again later" button would actually be the less irritating choice.

3) Convert the persistent pop-up to an available alert / message in the upper right corner, where a user who wants to continue using Apple Media Services can easily find it. 

In this case? I fumed for a while -- then I caved. I googled for ways to just make it go away, but none seemed worth it. I eventually dug up my Apple ID password, for which I otherwise have no use. The result is a better experience, but the hassle sticks in the craw, and a reminder of why I prefer open source environments.
